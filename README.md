# Xylix OS

Personal AI Operating System.

## Setup

1. Run `supabase-setup.sql` in your Supabase SQL Editor
2. Copy `.env.local.example` to `.env.local` and add your keys
3. `npm install`
4. `npm run dev`

## Deploy to Vercel

1. Push to GitHub
2. Import in Vercel
3. Add environment variables
4. Deploy
